## 공통 사용 가이드

### 플레이스홀더 치환 목록

| 플레이스홀더 | 내용 | 비고 |
|-------------|------|------|
| `{text}` | STT 변환 녹취 텍스트 | 전 양식 공통 필수 |
| `{dt}` | 생성 일시 (예: 2026-04-14 14:30) | 전 양식 공통 필수 |
| `{innoforest}` | 혁신의숲 API 조회 데이터 | IR 미팅회의록 전용 |

### 앱 적용 방법 (Android Kotlin 예시)

```kotlin
// 플레이스홀더 치환
val prompt = TEMPLATE
    .replace("{text}", sttText)
    .replace("{dt}", dateTimeString)
    .replace("{innoforest}", innoforestData ?: "(조회 데이터 없음)")
```

## 공통 작성 원칙 (전 양식 적용)

1. **팩트 기반**: 대화에서 실제 나눈 내용에만 기초하여 작성. 녹취에 없는 내용 절대 추가 금지
2. **Q&A 원칙**: Q&A로 실제 오고간 내용만 기록. 임의 추정하여 내용 추가 금지
3. **데이터 정밀도**: 수치·금액·일자 정확히 기록. 불확실한 경우 `(추정)` 표기
4. **Q&A 형식**: Q와 A는 각각 별도 blockquote(`>`) 줄로 작성. Q↔A 사이 빈 줄 1줄 삽입
5. **문체**: 모든 서술 문장은 "~음" 종결 (IR 미팅은 "~임/~함/~됨" 종결)
